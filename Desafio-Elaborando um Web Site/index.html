Tags <h1> a <h6>

Voltar

Com base no conteúdo do curso eu pude aprender que as tags h1 até h6 são usadas para definir títulos e subtítulos em HTML, organizando de forma hierárquica e semântica. Exemplo: a tag h1 é usada para o título mais importante e h2 a h6 são usadas para subtítulos de importância decrescente.

Tag<mark>

Voltar

Com base no conteúdo do curso eu pude aprender que a tag mark em HTML é usada para destacar trechos de texto, como se estivesse marcando com um marca-texto amarelo.

Tag<small>

Voltar

Com base no conteúdo do curso eu pude aprender que a tag <small> em HTML é usada para exibir texto em tamanho reduzido. Normalmente utilizada em Avisos legais; Notas de rodapé e Observações secundárias.

Tag<i>

Voltar

Com base no conteúdo do curso eu pude aprender que a tag <i> é usada para destacar texto com um estilo itálico.

Tag <u>

Voltar

Com base no conteúdo eu pude aprender que a tag <u> em HTML é usada para sublinhar um trecho do texto.

Tag <strong>

Voltar

Com base no conteúdo do curso eu pude aprender que a tag <strong> em HTML é usada para dar ênfase forte a um trecho do texto. Ela exibe o texto em negrito por padrão.

Tag <ol>

Voltar

Com base no conteúdo do curso eu pude aprender que a tag <ol> em HTML é usada para criar listas ordenadas, sendo que cada item é numerado automaticamente. Exemplo:

  1. Café
  2. Açúcar
  3. Pão francês integral
  4. Goiabada

Tag <li>

Voltar

Com base no conteúdo do curso eu pude aprender que a tag <li> em HTML é usada para definir um item dentro de uma lista, seja ela ordenada ou não.

Tag <a>

Voltar

Com base no conteúdo do curso eu pude aprender que a tag <a> em HTML é usada para criar links, ou seja, âncoras que conectam uma parte do conteúdo a outra página, a um arquivo, a um endereço de e-mail ou até uma seção da mesma página. Exemplo: Visite meu Linkedin

Tag <hr>

Voltar

Com base no conteúdo do curso eu pude aprender que a tag <hr> em HTML é usada para inserir uma linha horizontal que representa uma mudança de tema ou separação de conteúdo dentro de uma página.

Tag <sub>

Voltar

Com base no conteúdo do curso eu pude aprender que a tag <sub> em HTML é usada para exibir texto em formato subscrito, ou seja, abaixo de linha base do texto. Exemplo: Fórmula da Água: H20

Tag<sup>

Voltar

Com base no conteúdo do curso eu pude aprender que a tag <sup> em HTML é usada para exibir texto em formato sobrescrito, ou seja, acima da linha normal do texto. Exemplo: Em número 1020 ou texto EU GOSTO DE ESTUDAR!

Tag <bolckquote>

Voltar
